📄 HTML5 기초

HTML5는 웹페이지의 구조를 정의하는 마크업 언어입니다.

주요 특징

  • 시맨틱 요소 (Semantic Elements) 지원
  • 멀티미디어 콘텐츠 네이티브 지원
  • 향상된 폼 요소
  • Canvas와 SVG 지원
  • 로컬 스토리지 지원

시맨틱 요소 예제

<header> - 페이지 또는 섹션의 헤더

<article> - 독립적인 콘텐츠

이것은 기사, 블로그 게시물 등의 콘텐츠를 나타냅니다.

<section> - 주제별 콘텐츠 그룹

관련된 콘텐츠를 논리적으로 그룹화합니다.

<footer> - 페이지 또는 섹션의 푸터

기본 HTML5 구조

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>페이지 제목</title> </head> <body> <!-- 웹페이지 콘텐츠 --> </body> </html>

🎨 CSS3 기초

CSS3는 HTML 요소들의 스타일과 레이아웃을 정의합니다.

주요 특징

  • Flexbox와 Grid 레이아웃
  • 애니메이션과 트랜지션
  • 그래디언트 배경
  • 반응형 디자인 (Media Queries)
  • 변환 (Transform) 효과

Flexbox 레이아웃 예제

display: flex를 사용하면 요소들을 유연하게 배열할 수 있습니다.

.container { display: flex; justify-content: space-between; align-items: center; gap: 15px; }

Grid 레이아웃 예제

display: grid를 사용하면 2차원 레이아웃을 만들 수 있습니다.

.grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; }

애니메이션 예제

@keyframes slideIn { from { opacity: 0; transform: translateX(-20px); } to { opacity: 1; transform: translateX(0); } } .element { animation: slideIn 0.5s ease; }

⚙️ JavaScript 기초

JavaScript는 웹페이지의 상호작용을 담당하는 프로그래밍 언어입니다.

주요 특징

  • DOM 조작
  • 이벤트 처리
  • 비동기 처리 (Promise, async/await)
  • 배열과 객체 메서드
  • 로컬 스토리지 활용

DOM (Document Object Model)

DOM은 HTML 문서를 프로그래밍 방식으로 조작할 수 있게 해줍니다.

// 요소 선택 const element = document.getElementById('myId'); const elements = document.querySelectorAll('.myClass'); // 요소 수정 element.textContent = '새로운 텍스트'; element.classList.add('active'); // 이벤트 리스너 추가 element.addEventListener('click', function() { console.log('클릭됨!'); });

변수와 데이터 타입

// 변수 선언 const name = '홍길동'; // 상수 let age = 25; // 변수 var city = '서울'; // 전역 변수 (권장하지 않음) // 데이터 타입 const number = 42; // 숫자 const text = '문자열'; // 문자열 const bool = true; // 불린 const arr = [1, 2, 3]; // 배열 const obj = {name, age}; // 객체

함수 정의

// 일반 함수 function add(a, b) { return a + b; } // 화살표 함수 const multiply = (a, b) => a * b; // 함수 호출 console.log(add(5, 3)); // 8 console.log(multiply(5, 3)); // 15
