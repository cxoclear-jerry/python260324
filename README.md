# 코스피200 편입종목 상위 데이터 크롤러 (GUI 버전)

이 프로젝트는 네이버 금융에서 코스피200 지수의 편입종목 상위 데이터를 크롤링하는 PyQt6 기반 GUI 애플리케이션을 제공합니다.

## 요구사항

- Python 3.6 이상
- PyQt6 라이브러리
- requests 라이브러리
- beautifulsoup4 라이브러리
- openpyxl 라이브러리

## 설치

```bash
pip install PyQt6 requests beautifulsoup4 openpyxl
```

## 사용법

스크립트를 실행하면 GUI 창이 열립니다.

1. "크롤링 시작" 버튼을 클릭하여 코스피200 편입종목 전체 200개의 데이터를 수집합니다.
2. 크롤링 완료 후 "엑셀로 저장" 버튼을 클릭하여 데이터를 엑셀 파일로 저장합니다.

```bash
python kospi200_top_stocks_crawler.py
```

## GUI 구성

- **크롤링 시작 버튼**: 크롤링을 시작합니다.
- **엑셀로 저장 버튼**: 크롤링 완료 후 데이터를 엑셀 파일로 저장합니다 (파일명: kospi200[YYYY-MM-DD].xlsx).
- **진행률 바**: 크롤링 진행률을 표시합니다 (20페이지, 5%씩 증가).
- **로그 텍스트 영역**: 크롤링 진행 상황과 결과를 실시간으로 표시합니다.
- **결과 라벨**: 최종 수집 개수를 표시합니다.

## 기능

- 페이징 처리: 20페이지에 걸쳐 모든 200개 종목을 수집
- 멀티스레딩: 크롤링을 별도 스레드에서 실행하여 GUI 응답성 유지
- 실시간 로그: 진행 상황을 텍스트 영역에 표시
- 엑셀 저장: openpyxl을 사용하여 데이터를 엑셀 파일로 저장
- 서버 부하 방지: 각 요청 사이에 1초 대기
- 에러 처리: 요청 실패 시 해당 페이지를 건너뜀

## 엑셀 파일 형식

저장된 엑셀 파일에는 다음 열이 포함됩니다:
- 종목명
- 현재가
- 전일비
- 등락률
- 거래량
- 거래대금(백만)
- 시가총액(억)
